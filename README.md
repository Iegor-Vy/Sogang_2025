# Sogang University

This repository contains my Sogang University teaching materials, organized by academic year and semester.

- `2025/Spring/`: course folders for Central Banking, Financial Institutions and System, and Mathematical Economics.
- `2026/Spring/`: course folders for Financial Institutions and System, Mathematical Methods for International Commerce, and Statistics for International Commerce.
- Inside each course: week-by-week lecture files (mainly `.Rmd` and rendered `.html` slides), with supporting assets such as `libs/`, images, and theme files.
